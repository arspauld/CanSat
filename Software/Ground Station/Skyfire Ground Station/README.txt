Skyfire Ground Station Setup:
	1. Insert XBee module into USB drive on laptop
	2. Open device manager and search for the port the XBee module is recognized as
	3. Open main.py and change the port in line <line #> to the exact name the XBee is recognized as in the device manager
	4. Open terminal
	5. Navigate to the Ground Station folder
	6. Run the file by typing "python main.py"

****There is a "frozen" instance of main.py that runs like an executable if the XBee is recognized as the same port as dictated by the program
